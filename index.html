(Un)Natural Language

This is an art software that analyses public project documents through an ecological perspective. It is an Eco-linguistic system that explores the intersection of economics, language, and machine learning, unveiling the underlying language ideology in many infrastructure documents through an ambitious attempt to capture those pro-growth and greenwash sentences. This project builds a homemade dataset and utilizes a fine-tuned model to analyse formal documents like BP (business planning), IPO (initial public offering), and EA (environmental assessment). As a result, the software will automatically return a text analysis report. This system is trying to shift the focus of upgrowth economics to postgrowth ecological economics through the alternative approach to deploying machine learning technology and a language perspective.
